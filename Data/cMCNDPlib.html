The congested Multicommodity Network Design Problem Instance Library

(cMCNDPlib)

 

This is a copy of the library containing the instances for the congested Multicommodity Network Design Problem (cMCNDP), accompanying the technical report:

Paraskevopoulos, D., Gurel, S. and Bektas, T. 2013. The congested multicommodity network design problem. APOLLO Technical Report.

Sadly the original source isn't accessible anymore so we reupload it here.

The format of each file is given below:

1st column

Node ID (Incremental number that indicates the number of the node)

2nd column

Initial Capacity (in units of flow)

3rd column

Incremental capacity (The capacity added each time a node is upgraded)

4th to 6th columns

Incremental costs for the respective 0.6, 0.7 and 0.8 settings

7th column

Value of parameter Di
